API Documentation

          

            
/api/auth/test-login - POST

            
Test login credentials against a target domain

            
{
  "domain": "https://example.com",
  "username": "admin",
  "password": "password123",
  "loginUrl": "https://example.com/login",
  "usernameSelector": "#username",
  "passwordSelector": "#password",
  "submitSelector": "button[type='submit']"
}


            
/api/preview/screenshot - POST

            
Capture a screenshot of a webpage

            
{
  "url": "https://example.com",
  "sessionId": "optional-session-id"
}


            
/api/tools/run - POST

            
Execute a security testing tool

            
{
  "tool": "nmap",
  "target": "192.168.1.1",
  "options": {}
}


            
/api/reports/create - POST

            
Create a security test report

            
{
  "testName": "Vulnerability Assessment",
  "domain": "example.com",
  "findings": ["XSS vulnerability", "Missing security headers"]
}
