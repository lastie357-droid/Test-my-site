📡 API Documentation

          

            
POST /api/auth/test-login

            
Test login credentials against Shabiki via fapi.shabiki.com directly.

            
{
  "phone": "254704897825",
  "password": "panel123"
}


            
POST /api/preview/screenshot

            
Capture a screenshot of a webpage using Puppeteer.

            
{
  "url": "https://www.shabiki.com",
  "sessionId": "optional-session-id"
}


            
POST /api/tools/run

            
Execute a security testing tool against a target.

            
{
  "tool": "nmap",
  "target": "shabiki.com",
  "options": {}
}


            
POST /api/reports/create

            
Create a security test report.

            
{
  "testName": "Shabiki Login Audit",
  "domain": "shabiki.com",
  "findings": ["Missing HSTS", "Weak password policy"]
}
